Migration 1: agent_audit table

- profile_id (uuid, foreign key to relocation_plans, indexed)

- phase (text) - "extraction" | "validation" | "research" | "enrichment" | "settling-in" | "chat"
- field_or_output_key (text) - field key for extraction, section key for research, task ID for settling-in

Migration 2: pre_departure_actions table
- id (uuid, primary key)
- profile_id (uuid, foreign key to relocation_plans, indexed)
- title (text) - e.g., "Apply for A1 certificate"

- weeks_before_move_start (integer) - when to start
